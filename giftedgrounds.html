Project Feature

Gifted Grounds

A "mobile mall" that pulls many independent stores into one interface, so shoppers do not bounce between tabs to compare or buy.

Overview

Gifted Grounds is my take on a mobile-first mall: one storefront that hosts multiple brands so people can browse, compare, and check out without hopping across sites.

Live Site

giftedgrounds.co.za

Goal

Remove friction. Centralize discovery, carts, and checkout. Give every brand a home while keeping the experience consistent and fast on mobile.

Key Highlights

  • Multi-store catalog with consistent UI
  • Unified cart and checkout flow
  • Delivery options surfaced early
  • Payment gateway handoff ready
  • Responsive, mobile-first layout

Tech Stack

HTML, CSS, JavaScript, PHP, MySQL

What It Shows

Product sense and empathy for shoppers: fewer clicks, clearer delivery info, and a checkout designed for real people on real phones. It also reflects how I like to work - practical, user-first, and shipping things that reduce hassle.
