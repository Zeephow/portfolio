Project Feature

Newspaper Web App

A digital publishing platform built for editorial teams that need flexible content management and clean public-facing news pages.

Overview

This newspaper web app was built to help a media client publish and manage news content through a custom admin environment while keeping the reading experience structured, modern, and easy to navigate.

Admin Experience

I made the admin panel flexible enough for editors to manage content with full CRUD functionality and to edit the newspaper layout freely using GrapesJS, giving the client more control without needing developer support for every update.

Key Highlights

  • Custom article management workflows with CRUD operations
  • GrapesJS-powered layout editing for the admin side
  • Structured publishing flow for news content
  • Readable frontend designed for fast content consumption
  • Database-backed content management with MySQL

Tech Stack

HTML, CSS, JavaScript, PHP, MySQL, GrapesJS
